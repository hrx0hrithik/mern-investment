# MERN Investment & Referral Platform

## Features
- User authentication (JWT)
- Investment management
- Daily ROI calculation
- Multi-level referral income
- Cron-based scheduled jobs
- React dashboard with charts

## Tech Stack
- Node.js, Express
- MongoDB, Mongoose
- React, Vite
- JWT Authentication
- node-cron

### Backend
